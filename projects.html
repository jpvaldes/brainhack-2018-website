BIDS-app-Based Searchlight Analysis Pipeline

  • Michael Hanke
  • michael.hanke@gmail.com
  • BIDS-app
  • PyMVPA

Decoding analysis using a traveling "searchlight" are widely used approach to map the availability of a given signal, and can also be used for a representational similarity analysis (RSA). This project aims to produce a BIDS-app for a PyMVPA-based searchlight analysis for decoding and RSA.
